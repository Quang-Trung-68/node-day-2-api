# Node Day 2 API

This is the backend API for the Node Day 2 project.

## Requirements

- Node.js
- npm

## Installation

1. Install dependencies:
```bash
npm install
```

## Usage

Run the server in development mode (watch mode):

```bash
npm run dev
```

The server will run at: `http://localhost:3000`

## API Endpoints

APIs are defined under the `/api` prefix.

## Project Structure

- `server.js`: Application entry point
- `src/`: Main source code
- `db/`: Database (JSON/File based)
- `utils/`: Utility functions
